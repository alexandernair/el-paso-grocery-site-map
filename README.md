# El Paso Grocery Site Selector

An interactive map-based tool to explore potential locations for future
Trader Joe’s or H-E-B stores in El Paso, TX using open GIS data.

## Tech Stack
- React
- MapLibre GL JS
- Node.js
- Postgres + PostGIS
- Python
